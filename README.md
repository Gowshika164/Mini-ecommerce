# Mini E-Commerce Application

This project was developed as part of **Internship Task 2**.

It is a full-stack e-commerce application where users can view products,
see product details, and add items to the cart.

---

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MySQL

---

## Features
- Product listing
- Product details page
- Add to cart functionality
- Backend API integration
- Dynamic product images

---

```txt
mini-ecommerce/

How to Run the Project

Backend

Frontend
